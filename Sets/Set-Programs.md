# ⭐ Tips

- Sets store **unique values only**.
- Sets are **unordered**, so indexing is not supported.
- Set elements must be **hashable (immutable)**.
- Membership testing using a set is very fast (`O(1)` average).
- Use sets whenever you need to remove duplicates or perform fast lookups.
